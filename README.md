# Real-Time Object Detection using YOLO26

This project implements a custom object detection model using **YOLO26** and transfer learning to detect **chairs, tables, and trash bins**. The model was trained on a custom annotated dataset and evaluated using standard object detection metrics, achieving an **mAP@50 of 0.859** and **mAP@50-95 of 0.639**. The repository includes the complete training pipeline, dataset configuration, and sample prediction results.
